发表于 2017-12-20 | 分类于 PHP

记录在Windows下搭建laravel5.5环境，这里总结一下，方便自己之后laravel环境的搭建。

Laravel 5.5 文档–使用 Laragon 在 Windows 中搭建 Laravel 开发环境

Laragon集成开发环境

下载laragon
下载之后，开始安装即可。一定要勾选 Auto virtual hosts这个选项，可以自动生成项目对应的域名。安装成功之后，自己可以去laragon的设置中配置。

配置Laragon

在常规配置中，配置好项目根目录路径和本地访问域名

非常方便地切换 PHP、Apache、MySQL 的不同版本，也可选择Apache或Nginx作为web服务器。默认的服务器端口号为80，MySQL数据库的端口号为3306，由于我本地的80和3306端口被占用，即改为8080和3308，当然对应的配置文件的端口号信息也要重新配置。
